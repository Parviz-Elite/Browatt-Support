# MehrsoftIntegration

MehrSoft integration module for Browatt after-sales workflows.

This module is intentionally not copied from `Brandiol-Automation`. It follows the same modular idea, but the exposed client methods are Browatt-specific and focused on MehrSoft `AfterSales_*` operations.

Current client methods:

- `getProductStatusBySerial`
- `getWarrantyMonths`
- `getWarrantySettings`
- `saveAfterSales`
- `logout`
